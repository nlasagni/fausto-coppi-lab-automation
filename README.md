## Prerequisites

* Java 1.8

## Getting Started

```bash
git clone https://github.com/nlasagni/fausto-coppi-lab-automation.git
cd fausto-coppi-lab-automation
```

## DevOps setup
```bash
gradlew build
```

## Run Reservation Microservice in interactive mode
```bash
gradlew run
```
